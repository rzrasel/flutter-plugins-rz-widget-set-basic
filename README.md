# flutter-plugins-rz-widget-set-basic

Flutter Plugins Rz Widget Set Basic

## Installation

Add the dependency to your `pubspec.yaml`:

```yaml
dependencies:
  rz_widget_set_basic:
    git:
      url: https://github.com/rzrasel/flutter-plugins-rz-widget-set-basic.git
      ref: main
```
